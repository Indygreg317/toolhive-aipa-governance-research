# Partnership-Safe Language

This guide keeps the AIPA governance overlay neutral, respectful, and non-competitive.

## Preferred phrases

- ToolHive-style MCP registry
- MCP governance platform
- MCP runtime platform
- compatible governance overlay
- complementary open governance layer
- independent research prototype
- governance context layer
- audit evidence model

## Avoid phrases

- better than ToolHive
- replacement for ToolHive
- Stacklok integration
- Stacklok partnership
- official ToolHive governance layer
- competing MCP platform
- commercial clone

## Default disclaimer

This repository contains an independent AIPA research prototype. It does not claim endorsement, approval, partnership, or official integration with Stacklok, ToolHive, or any other MCP platform.

## Recommended summary

This work explores how governance records, Knowledge Blocks, execution receipts, and verification boundary maps could complement MCP-based tool control by making agent tool use easier to review and audit.
